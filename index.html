About this lab

  
A teaching tool for OWASP LLM01: Prompt Injection. Three scenarios:

  
    
    
  1. Direct injection — make a customer-service bot leak its hidden persona/secret.
    2. 
    
  2. Indirect injection — plant an instruction inside an email and watch the agent obey it (sending unauthorized email, etc.).
    3. 
    
  3. Data exfiltration — chain a fetched web page into the agent to leak secrets via a rendered markdown image URL.
    4. 
  

  
Use Mock mode for a deterministic classroom demo (no internet/API needed). Use Live mode with a real LLM to show that production models exhibit the same vulnerabilities.

  
All API keys are stored locally in your browser. Nothing is sent to the lab author.
